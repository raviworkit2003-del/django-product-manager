# How to Run Locally

- Clone the repo
git clone https://github.com/your-username/django-product-manager.git

- Go into project =
cd django-product-manager

- Create virtual environment =
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

- Install Django =
pip install django

- Run migrations =
python manage.py migrate

- Start server =
python manage.py runserver
